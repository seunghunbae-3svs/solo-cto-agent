# Cowork Public Skills

This folder contains public, generic skills intended for broad use. These are safe to ship in the npm package and do not include any private business context.

## Included skills
- cowork-cto: CTO-grade review and decision support
- cowork-ship: release readiness and rollback checks
- cowork-self-evolve: failure pattern capture and prevention
- cowork-advisor: strategy and prioritization guidance
- cowork-pitch-writer: concise pitch drafting
- cowork-idea-critic: constructive devil's-advocate critiques
